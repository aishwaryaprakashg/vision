# Privacy & Safety

## Privacy

VisionAI Studio is designed with a privacy-first approach.

All AI processing is performed locally within the user's browser using TensorFlow.js and Tesseract.js.

No uploaded images, extracted text, or analysis results are transmitted to any external server.

## Data Storage

The application stores only analysis history in the browser using LocalStorage.

No personal data is collected.

No analytics or tracking services are used.

## Permissions

The application only requires:

- Access to images selected by the user
- Browser storage (LocalStorage)

No camera, microphone, or location permissions are required.

## Safety

- Images remain on the user's device.
- AI predictions should be considered assistive and may not always be accurate.
- OCR quality depends on image resolution and clarity.
- Users should verify AI-generated results before making important decisions.

## Security

- No backend server
- No cloud AI inference
- No user authentication
- No external database
